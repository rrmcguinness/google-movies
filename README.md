# Google Movies Demo

## Setup

### Install Node JS

```shell
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash

# If you use bash
source ~/.bashrc

# If you use zsh
source ~/.zshrc

# Install Node JS 20 (LTS)
nvm install 20

# Test
node -v
```

### Run the demo

From the terminal, open the directory.

```shell
cd ~/Projects/google-movies

npm install

npm run dev
```

Click on the link in the terminal.
